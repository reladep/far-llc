## Key Takeaways for FAR

### UI/UX Must-Haves
1. **Trust signals** — Professional design, credentials, "fiduciary" emphasis
2. **Search is primary** — Users come to search/browse, not browse casually
3. **Quick info → Detail** — Card shows key stats, profile page has depth
4. **Comparison native** — Side-by-side is a core feature
5. **Mobile-first** — Many users will browse on mobile

### Patterns to Adopt
- Search bar prominent on homepage
- Filter sidebar (fees, AUM, specialties, location)
- Firm cards with key stats (AUM, fees, years in business)
- "Save/Compare" functionality
- Clear CTAs (Request Info, Schedule Call)
