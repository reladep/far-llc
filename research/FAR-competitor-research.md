# FAR, LLC — Competitor / comparable landscape (living doc)

_Last updated: 2026-02-11_

## FAR positioning (working)
A database + analysis layer for **wealth management firms** that helps people **find, vet, and compare** firms (fees, growth metrics, updates, etc.).

## What counts as a “competitor” (buckets)
1) **Advisor/firm discovery & matching (consumer-facing)** — helps users find an advisor/firm.
2) **RIA/firm data platforms (B2B / industry)** — Form ADV-derived databases, firmographics, contact data, growth signals.
3) **Rankings & editorial lists** — useful for traffic + credibility; sometimes interactive.
4) **Regulatory + public data sources** — baseline truth sources (not competitors but inputs).
5) **Review / lead-gen marketplaces** — may monetize via lead sales.

### 4) Regulatory / public data sources (inputs)
- **SEC Investment Adviser Statistics** (market sizing + counts; potential baseline reference).
  - Mentioned via AdvizorPro article linking to SEC: https://www.sec.gov/data-research/statistics-data-visualizations/investment-adviser-statistics

## FAR research worksheet (recommended columns)
For each comparable platform, capture:
- Name + URL
- Primary user (consumer / advisor / asset manager / recruiter / vendor)
- Core value prop
- Data sources (Form ADV, surveys, scraping, partnerships)
- Pricing model (subscription / lead fee / listing fee / freemium / ads)
- Coverage depth (firm-level, advisor-level, both)
- Comparison features (fees, AUM, growth, specialties, custody, etc.)
- Freshness / update cadence
- SEO strategy (programmatic pages? editorial?)
- Notes (design patterns, onboarding, trust signals)
